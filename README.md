# GPUI DEB Installer 📦

一个使用 [GPUI](https://www.gpui.rs/) 框架编写的高性能、轻量级图形化 `.deb` 软件包安装器。

![Screenshot](icon.svg)

## 特性 ✨

- **快速流畅**：基于 GPU 加速的 GPUI 渲染引擎，响应极其迅速。
- **现代化 UI**：简洁直观的界面，支持实时安装/卸载进度显示。
- **全面信息展示**：支持查看包名、版本、描述、维护者、安装大小等元数据。
- **依赖查看**：独立的 **Dependencies** 标签页，以清晰的列表展示软件包依赖。
- **已安装管理**：**Installed** 标签页可浏览并搜索系统中已安装的软件包。
- **系统集成**：支持通过命令行参数启动，可直接在文件管理器中右键打开 `.deb` 文件。
- **权限安全**：使用 `pkexec` 调用系统软件包管理器，确保操作安全受控。

## 系统要求 🛠️

- **操作系统**：Linux (Ubuntu, Debian, Deepin, UOS 等基于 Debian 的发行版)
- **核心依赖**：`dpkg`, `apt`, `policykit-1` (pkexec)
- **编译依赖**：Rust 工具链, `libvulkan-dev`, `libssl-dev`, `pkg-config`, `libwayland-dev` (如果您在 Wayland 下)

## 快速开始 🚀

### 1. 编译与运行

首先确保您已安装 Rust。

```bash
# 克隆仓库
git clone https://github.com/your-username/gpui-deb-installer.git
cd gpui-deb-installer

# 运行 (不加载文件)
cargo run --release

# 运行 (加载指定 deb 文件)
cargo run --release -- /path/to/your/package.deb
```

### 2. 桌面环境集成 (开发模式)

本工作区提供了一个 `install-icon.sh` 脚本，可将应用右键菜单和图标集成到您的本地桌面系统中，而无需安装到 `/usr/bin`。

```bash
chmod +x install-icon.sh
./install-icon.sh
```

集成后，您可以直接在文件管理器（如 Nautilus, Thunar 或 Dolphin）中**右键点击 `.deb` 文件**并选择 **"DEB Installer"**。

### 3. 构建发布 DEB 包

```bash
# 安装 cargo-deb
cargo install cargo-deb

# 构建 .deb 包
cargo deb
```

生成的安装包将位于 `target/debian/gpui-deb-installer_*.deb`。

## 项目结构 📂

- `src/main.rs`: 应用入口与参数解析
- `src/app.rs`: 主窗口视图与标签页切换逻辑
- `src/views/`:
  - `install.rs`: 安装逻辑、deb 文件信息展示与实时日志
  - `packages.rs`: 已安装软件包列表与卸载逻辑
- `src/models/`: 数据模型与本地 SQLite/JSON 缓存（如果适用）
- `src/utils/`: 封装 `dpkg` 命令行调用与 deb 文件解析
